# Founder Metrics

## Weekly Health KPI Set

- Active clients and conversion throughput
- Delivery risk count (`BLOCKED` + overdue)
- Revenue vs expenses this month
- Outstanding invoices and payout obligations
- Freelancer utilization and overload flags
- Upcoming events and expected revenue
- Content output by platform and status

## Data Rules

- All KPIs trace back to one or more primary records
- Every metric update should produce an `Activity` event
- Automation outcomes should be measurable by completion and failure counts
